This project is a web-based BMI (Body Mass Index) Calculator developed using HTML for structure, CSS for styling, and JavaScript for functionality. Users can input their height and weight to instantly calculate their BMI. The application also categorizes the result into standard health ranges such as Underweight, Normal weight, Overweight, and Obese.
